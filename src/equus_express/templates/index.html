Welcome to Equus Express IoT Server

This is the landing page for your Secure IoT API Server. From here, you can explore various functionalities and endpoints.

  • API Documentation (ReDoc)
  • Health Check

Getting Started:

To interact with the API, you'll typically use an IoT device (like the equus_express.client.DeviceAgent). Devices will register themselves and send telemetry data to this server.

You can find more information in the API documentation or by referring to the project's README.
